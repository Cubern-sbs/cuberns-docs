<Info>
  This class inherits `ActiveInstance` which means that all properties, functions and signals from ActiveInstance can be used.
</Info> 
## Properties
#### `Intensity: float`
Defines the intensity of the sun

#### `MaximumDistance: float`
The maximum distance till the suns light will have
#### `Shadows: bool`
If shadows are enabled or disabled
#### `LightColor: Color3`
The color for the suns light
