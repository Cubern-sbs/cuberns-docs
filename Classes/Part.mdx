## Properties
#### `name: string`
Returns the name of the `Instance`

#### `shape: string`
Returns the shape of `Instance`

#### `transparency: float`
Returns transparency of `Instance`

#### `collisions: bool`
Returns if collisions are allowed or not.

#### `anchored: bool`
Returns if `Instance` is anchored or not.

#### `position: Vector3`
Returns the position of `Instance`

#### `scale: Vector3`
Returns the scale of `Instance`

#### `rotation: Vector3`
Returns the rotation of `Instance`

#### `color: Color3`
Returns the color of `Instance`

#### `material: Material`
Returns the material of `Instance`

## Functions
#### `FindFirstChild(string name)`
Returns the child with the name given in parameter else it returns `Nil`. An example:
```lua
Map["Cube"]:FindFirstChild("cube1").anchored = true
```

#### `IsA()`
Returns a string of class name. An example:
```lua
printl(Map["Cube"].IsA())
```

#### `Destroy()`
Destroys the `Instance`. An example:
```lua
Map["Cube"]:Destroy()
```

## Signals
#### `OnBodyEntered`
This signal is called when a Body enters the `Instance` or touches it. An example:
```lua
Map["Cube"]:Connect("OnBodyEntered", __SCRIPT__, "FunctionToCall")

function FunctionToCall(Body)
    if Body.IsA() == "Player" then
        printl("Its a player!")
    end
end
```

#### `OnBodyExited`
This signal is called when a Body leaves the `Instance` or stops touching it. An example:
```lua
Map["Cube"]:Connect("OnBodyExited", __SCRIPT__, "FunctionToCall")

function FunctionToCall(Body)
    if Body.IsA() == "Player" then
        printl("Player! left")
    end
end
```

#### `OnMouseEntered`
This signal is called when a mouse hovers over the `Instance`. An example:
```lua
Map["Cube"]:Connect("OnMouseEntered", __SCRIPT__, "FunctionToCall")

function FunctionToCall()
    printl("Mouse entered!")
end
```

#### `OnMouseExited`
This signal is called when a mouse leaves over the `Instance`. An example:
```lua
Map["Cube"]:Connect("OnMouseExited", __SCRIPT__, "FunctionToCall")

function FunctionToCall()
    printl("Mouse left!")
end
```
