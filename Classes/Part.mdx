<Info>
  This class inherits `ActiveInstance` which means that all properties, functions and signals from ActiveInstance can be used.
</Info>

## Properties
#### `shape: string`
Returns the shape of `Instance`

#### `transparency: float`
Returns transparency of `Instance`

#### `collisions: bool`
Returns if collisions are allowed or not.

#### `anchored: bool`
Returns if `Instance` is anchored or not.

#### `color: Color3`
Returns the color of `Instance`

#### `material: Material`
Returns the material of `Instance`

## Functions
#### `IsA()`
Returns a string of class name. An example:
```lua
printl(Map["Cube"].IsA())
```
