<Info>PluginServices will refuse to handle/process any function or property change until unless a plugin window is not built. </Info>
## Functions 
#### `BuildWindow(string window_title, bool maximizable = false, bool always_on_top = false)`
The `BuildWindow` function will build the plugin window.
