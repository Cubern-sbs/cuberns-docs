## Constructors
#### `Vector2.new(x, y)`
`x` and `y` must be integers! An example:
```lua
local vector2 = Vector2.new(2, 4)
printl(vector2.x) -- prints 2
```

## Properties
#### `x`
The property x component of the Vector2

#### `y`
The property y component of the Vector2
